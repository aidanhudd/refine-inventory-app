# Refine Inventory Estimator - Upgraded Starter

This version adds:
- category dropdown from Supabase
- quantity type dropdown from Supabase
- quantity, SKU, location, notes, status
- search and filters
- inventory stats

## Before deploying
Make sure your Supabase tables have data:

### categories
Examples:
- Quartz
- Granite
- LVP
- SPC
- WPC
- Cabinets
- Tile / Hardwood

### quantity_types
Examples:
- slab
- piece
- box
- sq ft
- bundle
- pallet

## Deploy
Upload these files to your GitHub repo root, then redeploy on Vercel.
